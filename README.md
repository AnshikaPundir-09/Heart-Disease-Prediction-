# Heart Disease Prediction System 

A machine learning project that predicts the possibility of heart disease using clinical parameters and Logistic Regression.

## Features
- Predicts heart disease risk
- Uses 13 clinical attributes
- Data preprocessing and visualization
- Model evaluation using performance metrics
- Accurate and efficient prediction system

## Parameters Used
- Age
- Sex
- Chest Pain Type
- Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Exercise Induced Angina
- And more...

## Machine Learning Model
- Logistic Regression

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy

## Evaluation Metrics
- Accuracy
- Precision
- Recall

## Workflow
1. Data preprocessing
2. Train-test split
3. Model training
4. Prediction and evaluation

## Applications
- Healthcare analytics
- Early disease prediction
- Clinical decision support systems

## Future Improvements
